SUMMARY OF RESULTS

Overview:
The SCHOLARITY platform prototype was developed to help students access educational resources, connect with tutors, and watch lecture videos in a single software system. The project followed the Software Development Life Cycle (SDLC) phases including Planning, Design, Testing, and Final evaluation.

Implemented Features:
- Dashboard showing user activity.
- Resource library where students can download learning materials.
- Tutor list page where students can view tutors and book sessions.
- Video lectures section for watching educational videos.
- Video player with lecture details and downloadable notes.

Testing Results:
User testing was conducted using a Google Survey form , distributed to different participants including students and tutors. The feedback showed that:
- Most users found the platform “Easy or very easy to navigate”.
-“Resource Download Feature and Learning Materials” were considered the most useful.
- Most participants confirmed that the system features worked properly.
- The overall user interface design received ratings between 4 and 5 out of 5.

User Feedback:
Some users suggested minor improvements such as:
- Improving the navigation experience
- Making the interface more visually attractive
- Adding more features for learning support

Conclusion:
The SCHOLARITY prototype successfully demonstrates a functional educational platform. Based on the testing results and user feedback, the system is considered effective and user friendly. Future improvements can focus on enhancing the design and adding additional educational tools.
